Basics of C++ Workshop

A workshop covering the fundamentals of an integral language in industry.

This is a short, five-segment workshop on the basics of C++ and its general uses. Created by William O'Rosky and compiled by Oneal Abdulrahim.

The only required materials for this workshop include: Text editor and C++ compiler (we recommend Visual Studio Code and GCC), or IDE (we recommend Visual Studio)
